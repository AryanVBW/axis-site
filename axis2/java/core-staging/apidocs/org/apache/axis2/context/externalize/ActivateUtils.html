org.apache.axis2.context.externalize

Class ActivateUtils

  • java.lang.Object
    • org.apache.axis2.context.externalize.ActivateUtils

  • public class ActivateUtils
extends Object
    Some Externalize objects must be "activated" after they are read. Activation normally involves associating the object with objects in the current runtime. ActivateUtils provides activation related utilities

      • getAxisServiceExternalizeExtraName

        public static String getAxisServiceExternalizeExtraName(AxisService axisService)
        Return a Sring that contains the service QName and port local name of an AxisService seperated by a delimiter. This value can be used as part of externalizing an AxisService to provide additional information during deserialization in cases where the AxisService name is not unique or does not match for whatever reasons.
        Parameters:
        axisService - The AxisService to create the externalized name
        Returns:
        a String with the ServiceQName and port local name separated by DELIMITER_SERVICE_PORT if both values exist as parameters on the service; null otherwise.
